# Portfolio Website

A premium, modern portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Modern Design**: Sleek dark mode with glassmorphism effects.
- **Responsive**: Fully responsive across all devices.
- **Interactivity**: Smooth scrolling and reveal animations.
- **Sections**:
  - Hero with dynamic greeting
  - About Me
  - Skills Grid
  - Project Showcases
  - Contact Form

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Getting Started
Simply open `index.html` in your browser to view the site.
